期货农场 · 7 日轮回

每交易日收盘按当日成交统计调价；第 7 天为交割日：在收盘调价之后，按全市场该品种多空持仓聚合出交割参考价。

  • 作物：使用公共现货池与你背包中的作物实物交割。先处理所有玩家的空单（手数大者优先），再处理多单；空单按可用现货交割，缺货部分按交割价10 倍赔偿；多单按池内余量提货，现货不足时可无货了结且无需付现，否则须付现提货入背包。交割时先扣款，扣完后现金< 0 才判游戏失败。
  • 种子：不参与作物现货池。空头按交割价做现金盈亏结算；多头按交割价付现，对应数量种子进入背包；同样先扣款，现金 < 0 才失败。
  • 交割结束后，仅作物挂牌价在整数百分比内随机波动；若本局出现过交割失败类情况，波动区间会放大。
  • 背包：种子可点击转化为作物现货；作物现货可按现价卖出，获得现金，等量实物回到公共现货池。
