期货星露谷 · 52 周 × 7 日

全局长度 52 周，每周 7 天；第 7 天交割日（先按设计结算期货与现货，再进入下一周）。开局欠债 200 万，从第 2 周起每周扣利息 3000。期货日终按事件因子 + 净需求因子（5 日均量平滑）得到临时变动率（±10% 截断），并与现货价按 0.8/0.2 收敛；开仓平仓手续费为成交额 0.5%（单笔 1～500 元）。

  • 现货与商店：公共现货池变化会推动现货价；商店买入价 = 现货×0.95、卖出价 = 现货×1.05。背包作物按商店卖出价卖给商店时，等量实物回池并影响次日相关输入。
  • 交割：按交割前快照的期货价与商店价结算；多头付现提货、空头交货或按商店价补货；若现金不足会强制按商店卖出价卖库存等。交割后净资产 = 现金 + 库存×商店买入价；< 0 判破产出局。
  • 创业板：支付 20 万开通后可交易稀有品种合约；做空有保证金与名义上限（含池子比例等规则）。
  • 农场：种子在地块上需生长天数成熟（非一点即得）；可升级土地、使用化肥、稀有产出等随版本扩展。
  • 借贷 / NPC / 长事件：局内提供借债与商人等入口；完整规则见仓库内《游戏设计》文档。

当前版本仅开放「本地单机」。创建/加入房间与跨设备联机已关闭。若需本地调试房间或联机，可在 URL 加 ?rooms=1，或将 localStorage 键 futures-game:enableRooms 设为 "1" 后刷新；与 ?mp=local 联用可仅用本地内存房间（不连 Playroom）。正式开放时由配置项 roomAndOnline 控制。
