# 投资公司发展物语 V2.0 开发路线图 v0.2

> 本文档记录当前已实现功能与待实现功能的清单，便于追踪开发进度与后续规划。

---

## 已实现功能

### v0.1 Demo 核心闭环

| 功能 | 状态 | 说明 |
|-----|------|------|
| 股票投资 | ✅ 已实装 | 支持单股/组合投资，10行业×2股共20支个股，按权重分配仓位 |
| 期货交易 | ✅ 已实装 | 4品种（综合/能源/金属/农产品），支持1x/2x/3x杠杆 |
| 月结结算 | ✅ 已实装 | 表驱动可复现结算，含大环境/行业/个股/能力/噪声五因子 |
| 业务利润策略 | ✅ 已实装 | 支持「滚存复利」与「上交公司」两种模式 |
| 业务透支处理 | ✅ 已实装 | 月结后账户为负时，可选择续资补足或清算结业 |

### v0.2 人事与AI增强

| 功能 | 状态 | 说明 |
|-----|------|------|
| 人才库系统 | ✅ 已实装 | 支付2万刷新，获得3~5名带AI风格的候选人 |
| 员工AI交易风格 | ✅ 已实装 | momentum/trend/dividend 三种风格，影响股票组合生成 |
| 季度AI调仓 | ✅ 已实装 | 每3个月自动按风格重建组合，满仓操作 |
| 股票轻仓机制 | ✅ 已实装 | 非AI员工或AI无法生成有效组合时，使用20%轻仓 |
| 月分红机制 | ✅ 已实装 | 成熟期股票按年股息率÷12计提月分红 |
| 月薪公式（工龄制） | ✅ 已实装 | 0.5万 + 工龄×0.1万 |
| 员工管理上限 | ✅ 已实装 | 初级50万/中级500万/高级1000万 |
| 月度报告弹窗 | ✅ 已实装 | 详细盈亏汇总与业务明细 |

### v0.3 阶段与多维能力

| 功能 | 状态 | 说明 |
|-----|------|------|
| 游戏阶段系统 | ✅ 已实装 | 初创期/扩张期/成熟期三阶段，触发条件+解锁内容+晋升弹窗 |
| 多维能力系统 | ✅ 已实装 | 领导力/创新力/执行力 + 8行业技术 |
| 咨询服务 | ✅ 已实装 | 当月完成，分行业选择，领导力+行业技术影响收益 |
| 拉取投资 | ✅ 已实装 | 多月周期（1-6月），领导力影响周期和成功率 |
| 结算公式更新 | ✅ 已实装 | 不同业务使用不同能力权重 |

### v0.4 股权与上市

| 功能 | 状态 | 说明 |
|-----|------|------|
| 公司股份系统 | ✅ 已实装 | 股权结构、创始人股份、员工期权池 |
| 上市准备流程 | ✅ 已实装 | 申请→准备→挂牌，含条件校验与阶段进度 |
| NPC投资者系统 | ✅ 已实装 | 天使/VC/PE三档，随机触发出价 |
| 增发股票 | ✅ 已实装 | 已上市公司可向市场增发募资 |
| 年报系统 | ✅ 已实装 | 每年1月自动生成年度财务报告 |
| 股价联动 | ✅ 已实装 | 上市后股价随公司月度净利润变化 |
| 股权稀释确认 | ✅ 已实装 | 拉投资≥50万时需确认出让股份比例 |
