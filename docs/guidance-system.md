# 指导系统 v0.1

> 本文档说明玩家对业务的「指导」机制，包括股票投资的风格/组合/资金调整，以及期货的杠杆调整。

---

## 1. 指导概述

### 1.1 基本概念

指导是玩家对某一在营业务的**干预手段**，每月限用1次（全公司共享）。通过指导可以：
- 调整股票投资的风格和组合
- 调整期货交易的杠杆倍数
- 对业务进行增资或减资

### 1.2 使用限制

| 限制项 | 规则 |
|-------|------|
| 次数 | 每月1次，全公司共享 |
| 对象 | 仅当月新开或已有的在营业务 |
| 互斥 | 本月被培训的员工不可接受指导 |
| 触发时机 | 业务布署后随时可用 |

---

## 2. 股票指导

### 2.1 可调整项

对股票业务，指导可修改以下1项或多项：

| 调整项 | 说明 | 可选值 |
|-------|------|-------|
| 投资风格 | 影响收益期望 | 保守/平衡/激进 |
| 组合权重 | 多股持仓分配 | 各股权重之和=100% |
| 资金调拨 | 增减业务本金 | 增资：从公司现金划入；减资：从业务划出至公司 |

### 2.2 投资风格

| 风格 | 期望收益修正 | 适用场景 |
|-----|-------------|---------|
| 保守 | -2%（万分比-200） | 低迷市场求稳 |
| 平衡 | 0（基准） | 常规操作 |
| 激进 | +4%（万分比+400） | 向好市场博取高收益 |

**结算公式中的位置**：
```
P = 大环境 + 行业 + 个股 + 能力 + 【指导因子】 + 噪声
```

### 2.3 组合权重

#### 默认组合
员工新开股票业务时，系统自动生成**前4支股票均分**的组合：

```
STK0001: 25%
STK0002: 25%
STK0003: 25%
STK0004: 25%
```

#### 自定义组合
通过指导可调整为任意组合，需满足：
- 至少持有1支股票
- 各股权重之和 = 100%（10000万分比）
- 仅可配置已上市股票（受`listingYearMonth`限制）

#### 组合因子的结算影响

```javascript
// 行业因子加权
portfolioSectorBp = Σ( weight_i × sectorBetaBp_i ) / 10000

// 个股因子加权  
portfolioBetaBp = Σ( weight_i × betaExtraBp_i ) / 10000
```

### 2.4 资金调拨

| 操作 | 条件 | 效果 |
|-----|------|------|
| 增资 | 公司现金充足 | 从公司现金 → 业务AUM；若策略为"上交"，同步增加initialWan |
| 减资 | 减资后AUM ≥ 1万 | 从业务AUM → 公司现金；若策略为"上交"，同步减少initialWan |

**限制**：
- 最低保留AUM：1万（防止业务过小结算异常）
- 上交策略下，减资后初始本金也必须≥1万

## 3. 期货指导

### 3.1 可调整项

| 调整项 | 说明 | 可选值 |
|-------|------|-------|
| 杠杆倍数 | 放大收益/亏损 | 1x / 2x / 3x |
| 资金调拨 | 同股票 | 增资/减资 |

### 3.2 杠杆机制

期货收益率在股票公式基础上，**乘以杠杆倍数**：

```
P0 = 品种因子 + 能力因子 + 噪声
P  = P0 × L   (L ∈ {1, 2, 3})
```

**风险警示**：
- 杠杆同时放大盈利和亏损
- 2x杠杆下，-5%的P0变成-10%的实际P
- 高杠杆容易导致业务透支

### 3.3 品种切换

新开期货业务时需选择品种，**结算后不可更改**。当前可选品种：

| 品种 | 特点 | 繁荣期收益 | 冰点期亏损 |
|-----|------|-----------|-----------|
| 综合 (composite) | 基准平衡 | +6% | -6% |
| 能源 (energy) | 高波动 | +8% | -8% |
| 金属 (metal) | 中波动 | +5% | -5% |
| 农产品 (agri) | 低波动 | +4% | -4% |

## 4. 指导操作流程

### 4.1 操作步骤

1. **选择业务**：在「在营业务」列表中选中目标业务
2. **点击指导**：进入指导面板
3. **调整参数**：
   - 股票：修改风格（可选）、调整组合权重（可选）、增减资金（可选）
   - 期货：修改杠杆（可选）、增减资金（可选）
4. **确认应用**：至少修改一项后，点击「应用指导」
5. **消耗次数**：本月剩余次数从1变为0

### 4.2 界面反馈

指导成功后日志输出示例：
```
【指导·股票】张三 风格2、组合（本月剩余 0 次）
【指导·期货】李四 3x、增资10万（本月剩余 0 次）
```

## 5. 策略建议

### 5.1 风格选择参考

| 市场行情 | 建议风格 | 理由 |
|---------|---------|------|
| 繁荣(🔥) | 激进 | +4%期望叠加高基础收益 |
| 向好(📈) | 激进/平衡 | 激进追求高收益，平衡稳健 |
| 平稳(➡️) | 平衡 | 基准期望，不冒进 |
| 低迷(📉) | 保守 | -2%修正可部分抵消负基础 |
| 冰点(💀) | 保守/平衡 | 冰点市场考虑不上业务 |

### 5.2 杠杆选择参考

| 市场行情 | 建议杠杆 | 预期年化波动 |
|---------|---------|-------------|
| 繁荣 | 3x | ±18% |
| 向好 | 2x~3x | ±12%~18% |
| 平稳 | 1x~2x | ±6%~12% |
| 低迷 | 1x | ±6% |
| 冰点 | 不建议开仓 | — |

### 5.3 资金调拨策略

**增资场景**：
- 业务盈利良好，追加投入扩大收益
- 业务接近透支，提前增资防范

**减资场景**：
- 市场转向悲观，减少敞口
- 回收资金用于其他业务或人事
- 注意：减资后AUM不得低于1万
